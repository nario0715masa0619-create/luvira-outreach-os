# KPI Design

本ドキュメントでは、Luvira Outreach OSにおけるKPI（重要業績評価指標）の設計思想と、フォーム営業・メール営業それぞれの評価指標、および月次レポートの構成を定義します。

## KPI Design Principles
本システムのKPI設計は、以下の原則に基づいています。
1. **分離と可視化:** 「アプローチ先が適切だったか（リスト適合性）」と「システムが正常に動作したか（実行成功率）」を厳密に分離し、問題の切り分けを可能にする。
2. **チャネル横断での比較:** メール営業とフォーム営業のパフォーマンスを同じ基準で比較・評価できるようにする。
3. **アクションへの直結:** 単なる数値の羅列ではなく、リストの改善、文面の改善、システムチューニングのどのアクションを取るべきかが明確になる指標を定義する。

## Form Outreach KPIs
フォーム営業特有の指標です。相手企業のサイト構造に依存するため、エラーの内訳が重要になります。

- **処理件数 (Processed):** CSVから読み込まれ、フォーム営業ルートに振り分けられた総件数。
- **リスト適合性 (List Suitability):**
  - **フォーム発見率:** 対象企業のWebサイト内に、問い合わせフォームが存在した割合。
  - 計算式: `(フォーム発見件数 / 処理件数) * 100`
- **実行成功率 (Execution Success Rate):**
  - **フォーム送信成功率:** 発見されたフォームに対して、正常に入力・送信が完了した割合。
  - 計算式: `(送信完了件数 / フォーム発見件数) * 100`
  - *失敗の主な内訳:* CAPTCHAブロック、必須項目の不一致、DOM構造の例外など。
- **アプローチ完了率 (Completion Rate):** 処理件数に対する、最終的な送信完了件数の割合。
  - 計算式: `(送信完了件数 / 処理件数) * 100`

## Email Outreach KPIs
メール営業特有の指標です。一般的なメールマーケティングの指標に加え、BtoB営業特有の到達性に焦点を当てます。

- **処理件数 (Processed):** メール営業ルートに振り分けられた総件数。
- **リスト適合性 (List Suitability):**
  - **有効アドレス率:** リスト内のメールアドレスが構文的に正しく、かつ存在している（バウンスしない）割合。
  - 計算式: `((処理件数 - ハードバウンス件数) / 処理件数) * 100`
- **実行成功率 (Execution Success Rate):**
  - **配信成功率:** エラーにならずに相手のメールサーバーへ到達した割合。
  - 計算式: `(到達件数 / 有効アドレス件数) * 100`
- **エンゲージメント指標:**
  - 開封率 (Open Rate)
  - リンククリック率 (CTR)
  - 返信率 (Reply Rate)

## Combined KPI View
両チャネルの総合的なパフォーマンスを評価するための統合指標です。

- **総合リスト投入件数**
- **チャネル別振り分け比率:** フォーム vs メールの比率。
- **総合アプローチ完了件数:** フォーム送信完了 + メール配信成功 の合算。
- **商談獲得率 (CVR):** 総合アプローチ完了件数に対する、実際の商談化やポジティブな返信の割合（※オフラインのデータ突合が必要）。

## Required Logs
上記KPIを正確に算出するため、システム（activity-logger）は各実行ステップで以下のログを記録します。

1. **実行ID (Execution ID):** トランザクション単位のユニークID。
2. **企業ID/名称:** 対象企業の識別子。
3. **チャネル:** `form` または `email`。
4. **ステータス:** `success`, `error`, `skipped` 等。
5. **エラーカテゴリ:** 失敗した場合の理由分類（例: `form_not_found`, `captcha_blocked`, `invalid_email`, `bounce`）。
6. **タイムスタンプ:** 処理の発生日時。

## Monthly Report Layout
月次レポートでは、上記の指標を以下の構成でダッシュボードまたはレポート形式で提示します。

1. **Executive Summary:** 今月の総合アプローチ件数、完了率、商談化件数（ハイライト）。
2. **Channel Performance:** フォーム営業とメール営業のKPIサマリ比較。
3. **Form Outreach Deep Dive:** フォーム発見率の推移、エラー原因の内訳（円グラフ等）、改善提案（Browser Useオプションの適用検討など）。
4. **Email Outreach Deep Dive:** 配信成功率、開封率・返信率の推移、文面A/Bテストの結果。
5. **List Quality Analysis:** 提供されたリストの適合性評価と、次回リスト作成に向けた条件のフィードバック。
6. **Next Actions:** Luviraからのシステム・運用面における次月の改善タスク。
