# Service Scope

本ドキュメントでは、Luvira Outreach OSの導入における標準提供範囲（Standard Plan）、オプション機能、個別開発範囲、および顧客とLuviraの責任分界点を定義します。

## Standard Plan
標準プランでは、営業自動化を最速で立ち上げるための基本セットを提供します。

- **要件ヒアリングとプロセスマッピング** (最大2回)
- **n8n環境への共通OS展開**
- **CSVアップロードフローの構築** (指定フォーマット1種)
- **標準チャネルの実装**
  - メール送信ワークフロー
  - Webフォーム送信ワークフロー (ベーシックなDOM解析・POST送信)
  - 両チャネルの分岐（ルーティング）設定
- **実行ログとKPI集計の初期設定**
  - 標準ログスキーマでのイベント記録
  - 汎用出力（例: Google Sheetsへの標準レイアウト出力）
  - 標準KPIレポートの提供
- **月次定例ミーティング** (1回/月) によるKPI報告と改善提案
- **稼働監視とシステム保守** (n8nワークフローに関するもの)

## Optional Features
企業の成長や高度な要件に合わせて、以下の機能を追加オプションとして提供します。

- **高度なログ出力・連携 (Output Adapters):**
  - 顧客独自DB向けの列変換やマッピング対応
  - 独自CSVレイアウトでのエクスポート
  - 社内APIやWebhookへの連携出力
  - Salesforce/HubSpotなどCRM/SFAの取り込み形式に合わせたデータ出力・書き戻し
- **AI文面生成:** 顧客のWebサイト情報や企業概要をLLMに解析させ、パーソナライズされた営業文面を動的に生成する機能。
- **Browser Use連携:** SPAやCAPTCHAが設定されているなど、標準のフォーム送信では対応が困難なサイトに対する、ブラウザ自動操作を用いた高度なフォーム入力機能。
- **追加チャネル連携:** LinkedInメッセージやTwitter(X) DMなどへのアプローチ自動化。

## Custom Scope
以下のようなケースは「共通OS」の範囲外となり、別途個別開発の対象（お見積り）となります。

- 特殊な基幹システムや社内独自データベースとの双方向な密結合連携
- お客様指定の極めて複雑な条件分岐や承認フローを伴う営業プロセスの実装
- 月間数十万件を超えるような超大規模配信に耐えうる専用インフラチューニング

## Customer Responsibilities
本サービスの円滑な運用のため、お客様（顧客企業）には以下の役割を担っていただきます。

- **n8n環境の契約・維持:** n8n Cloudの契約、または自社インフラでのn8nホスティング費用と権限の提供。
- **営業リストの準備:** ターゲットとなる企業リスト（CSV等）の定期的な用意とアップロード。
- **ドメイン・アカウントの用意:** メーラーのSMTP情報や、送信元ドメインの用意（SPF/DKIM設定等）。
- **ビジネス要件の最終判断:** ターゲット選定、アプローチ文面、オファー内容などの営業戦略上の意思決定。

## Luvira Responsibilities
Luviraは以下の役割と責任を負い、お客様の営業活動をシステム・運用の両面から支援します。

- **ワークフローの安定稼働:** Luvira Outreach OSとして提供したn8nワークフローの正常な実行、エラーハンドリング、およびバグ修正。
- **アップデートの提供:** 共通OSにおける汎用的な機能追加やバグフィックスの顧客環境への適用。
- **データに基づくインサイト提供:** 標準ログから集計されたKPIレポートの作成と、それに基づくリスト選定・文面などの改善提案。
- **技術サポート:** OS利用に関するQ&A対応およびトラブルシューティング。

※「ログ項目の定義（OSとして何のデータを取得するか）」と「出力形式の個別最適化（どこに・どのような形でお客様に渡すか）」は明確に別物として扱い、前者は共通OSのアップデート範囲内、後者はオプションまたは個別開発のスコープとなります。
